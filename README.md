## ⚙️ 사전 준비 프로그램
팀원들이 프로젝트를 실행하기 위해 필요한 프로그램들:

1. **Windows Subsystem for Linux 2 (WSL2)**
   - Microsoft Store에서 Ubuntu 설치
   - 최신 버전으로 업데이트 필요

2. **Docker Desktop**
   - Windows에서 실행
   - WSL2 기반으로 동작
   - NVIDIA Container Toolkit 연동 필요

3. **NVIDIA GPU 드라이버**
   - CUDA 지원 GPU 환경 필요
   - `nvidia-smi`로 정상 작동 확인

4. **Python 3.10+**
   - `requests`, `openai` 패키지 설치 필요
